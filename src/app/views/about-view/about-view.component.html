About

Do you like aligning all those columns in your ZMK  keymap files? Well I know I didn't, and that is why I started the keymapper.dev webapp.

The application lets you graphically order the keys so that the layout on screen more or less resembles the physical layout of the keyboard. Then you can define the layers that you need and assign a behavior to each key in each layer. As a last step it is possible to define combos that let you add a behavior to a combination of simultaneously pressed keys. The .keymap file will be generated automatically but it is possible to apply your own 1337ness in the integrated code editor.

When the code is to your liking just copy it to the clipboard or download it to a local file to further process it in your ZMK project.

The menu lets you import or export the configuration in the form of a .json file, to facilitate sharing.
